## Overview

This project is a **static website for FHSU Movie Theater**. Users can:

- Browse movies currently showing and coming soon.
- View detailed movie pages with descriptions and showtimes.
- Select seats from a 5x5 grid.
- Proceed to a payment form for ticket purchase (simulated).

---

## Features

- **Homepage (`index.html`)**: Hero section, Now Showing, Coming Soon, and navigation.
- **Browse Page (`browse.html`)**: Search bar and grid of all movies.
- **Movie Detail Pages** (`springsteen.html`, `blackphone.html`, `goodfortune.html`, `tronares.html`): Movie info and showtime buttons.
- **Seat Selection / Payment (`pay.html`)**: 5x5 seat grid and payment form.
- **Responsive Design**: Layout adjusts for mobile and desktop.

---

## How to Run

1. Download or clone the project folder.
2. Open `index.html` in a web browser.
3. Navigate through pages by clicking movie cards and showtime buttons.
4. Select seats on the payment page and fill out the payment form.

> **Note:** This is a static website; form submission and search are not connected to a backend.

---

## Technologies

- HTML5
- CSS3 (style.css, browse.css, payment-style.css, seats.css)
- JavaScript (seat selection)
